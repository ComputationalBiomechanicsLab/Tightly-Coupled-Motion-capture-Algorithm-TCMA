# Tightly-Coupled-Motion-capture-Algorithm-TCMA
TCMA is an IEKF-based sensor fusion algorithm that tightly couples IMU measurements with multibody dynamic models in OpenSim to simultaneously estimate joint kinematics and kinetics. It supports fusion of IMUs, optical markers, and zero-torque updates, and is validated on a 3-DoF pendulum and a 6-DoF KUKA robot.
